================================================================================
                        QUICK START GUIDE
================================================================================

OPTION 1: Run Locally (Recommended First)

1. Open terminal in project folder
2. Run: ./setup-local.sh
3. Copy: cp .env.local.example .env.local
4. Edit .env.local with your credentials
5. Setup database (see SUPABASE_SETUP.md)
6. Run: pnpm dev
7. Open: http://localhost:3000

Detailed guide: See RUN_LOCALLY.md

OPTION 2: Deploy to Vercel

1. Push code to GitHub
2. Go to vercel.com/new
3. Import your GitHub repo
4. Add environment variables
5. Click Deploy

Detailed guide: See DEPLOYMENT.md

================================================================================
                        ENVIRONMENT SETUP
================================================================================

You need these accounts and values:

1. Supabase (Free)
   - Create project at supabase.com
   - Get: Project URL, Anon Key, Service Role Key

2. Stripe (Free)
   - Create account at stripe.com
   - Get: Public Key, Secret Key, Webhook Secret

3. Vercel (Free)
   - For deployment only
   - Free tier is perfect for testing

See .env.local.example for all variables needed
